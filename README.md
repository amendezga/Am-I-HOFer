# Am-I-HOFer
Do you want to know if your favorite player is good enough to make the hall of fame? You'll never have to guess again. Am I HOFer will allow you to search your desired player, and will analyze their career stats to let you know their likelihood to make the NFL Hall of Fame. 

## Run locally

This repo includes a minimal React app under `web/`.

```bash
cd web
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://127.0.0.1:5173/`).
